Solution Package Description

The solution directory contains all the code required to apply the model (for information on how to deploy and run the model see the corresponding pages). Note that the following is a maximal list, and some of the listed files or folders might not be available in your solution (depending on the workflow configuration).


    +-- <candidate plan id>
    |   +-- install_solution.sh                  # Solution installation.
    |
    |   +-- batch_client.py                      # Batch client.
    |   +-- online_client.py                     # Online client.
    |   +-- forecast_client.py                   # Forecast client.
    |   +-- Solution.ipynb                       # Jupyter Notebook.
    |
    |   +-- solution_server/                     # Server package that can be used to serve a
    |                                            # REST API for the solution.
    |   +-- solution_server_config.yml           # Default config file for the solution server.
    |   +-- solution_example_clients/            # Example clients to send requests to the solution
    |                                            # server.
    |
    |   +-- modulos_utils/                       # Directory that contains all utility and
    |                                            # helper functions to process the data.
    |
    |   +-- src/                                 # Directory that contains the trained weights,
    |                                            # the code of the model, configuration files
    |                                            # for the solution code, etc.
