API Options for the Solution

There are several options for how to use this solution to generate predictions. Depending on the workflow settings, some of them are available or not. Please see the individual sections for more details and information on which options are available.

We divide the different options into two categories: A prediction can be made either by deploying a server that serves a REST API and then sending a request to it, or by directly running the python prediction scripts (or the jupyter notebook) that are included in the solution.

  • Python prediction scripts / jupyter notebook
  • REST API (available for all workflows except for time series workflows)
